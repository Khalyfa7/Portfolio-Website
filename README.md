## Features

The Portfolio Website includes several key features:

- **Responsive Design:** The layout adjusts to different screen sizes, ensuring a great experience on mobile and desktop devices.
  
- **Accessibility Compliance:** The website follows best practices for accessibility, making it usable for all individuals.
  
- **Performance Metrics:** The site is optimized for speed and efficiency, with measurable performance metrics to demonstrate its capabilities.

- **Interactive Animations:** Using Framer Motion, the site features smooth transitions and animations that enhance user engagement.

- **Dynamic Content:** The site fetches data from a REST API, allowing for dynamic updates and content management.

## Installation

To set up the Portfolio Website locally, follow these steps:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Khalyfa7/Portfolio-Website.git
   ```

2. **Navigate to the Project Directory:**
   ```bash
   cd Portfolio-Website
   ```

3. **Install Dependencies:**
   ```bash
   npm install
   ```

4. **Set Up Environment Variables:**
   Create a `.env` file in the root directory and add your environment variables. 

5. **Start the Development Server:**
   ```bash
   npm start
   ```

6. **Build for Production:**
   To create a production build, run:
   ```bash
   npm run build
   ```
